# Project Description

a digital assistent for android and ios phones that will record conversations to later send to AI assistants to be processed.

# Functionality

- Simple UI that starts "taking notes" by transcribing voice from the selected input device.
- The transcription uses speaker identification APIs to annotate who is speaking.
- The user can select which applcations to share the transcription with when note taking stops or periodically during note taking

# Implementation

- Implemented using Kotlin Multi-Platform for Android and iOS.
